**Q: Can I deploy PRISM locally?**
A: Yes! SQLite version runs locally. Docker container coming soon.

**Q: Is there an API?**
A: JSON endpoints exist for visualization data. Full REST API in Phase 4.

**Q: How do you prevent spam/bad data?**
A: Phase 2 adds authentication. Phase 3 adds admin review workflows.

**Q: Can I bulk import my institution's tools?**
A: Yes! CSV upload supports enriched metadata. Contact us for help.

**Q: How is this different from bio.tools?**
A: We focus on cross-domain tool *interactions*, not just tool listings.

**Q: Will you integrate with EOSC/FAIRsharing?**
A: Yes, we're exploring data exchange partnerships.

**Q: What license is PRISM under?**
A: Code will be open source (MIT/Apache). Data is CC-BY 4.0.

**Q: How accurate is the AI discovery system?**
A: Human review is required. AI provides ~70-80% confidence suggestions.
